This example shows a strategy form configuration for the Price / Bollinger Bands formula. It compares the Close price from the current period (numerator) with the Upper Bollinger Band from 5 periods ago (denominator). The comparison operator is set to ">" (greater than), indicating that the strategy is looking for situations where the current Close price is higher than the Upper Bollinger Band from 5 periods ago. The Bollinger Bands parameters are set to use a 20-period SMA and 2 standard deviations.
