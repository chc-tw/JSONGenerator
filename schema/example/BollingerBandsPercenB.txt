This example shows a strategy form configuration for the Percent B (%B) Comparison formula. It compares the latest %B value (numerator) with the %B value from 5 periods ago (denominator). The comparison operator is set to ">" (greater than), and the compareValue is set to 5, indicating that the strategy is looking for situations where the latest %B is 5% higher than the %B 5 periods ago. The Bollinger Bands parameters are set to use a 20-period SMA and 2 standard deviations.
