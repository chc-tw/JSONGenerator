This example shows a strategy form configuration for the Bandwidth Comparison formula. It compares the latest Bandwidth (numerator) with the Bandwidth from 5 periods ago (denominator). The comparison operator is set to ">" (greater than), and the compareValue is set to 5, indicating that the strategy is looking for situations where the latest Bandwidth is 5% higher than the Bandwidth 5 periods ago. The Bollinger Bands parameters are set to use a 20-period SMA and 2 standard deviations.
