This example shows a strategy form configuration for the Bollinger Bands Comparison formula. It compares the latest Upper Band (numerator) with the Upper Band from 5 periods ago (denominator). The comparison operator is set to ">" (greater than), and the compareValue is set to 3, indicating that the strategy is looking for situations where the latest Upper Band is 3% higher than the Upper Band 5 periods ago. The Bollinger Bands parameters are set to use a 20-period SMA and 2 standard deviations.
